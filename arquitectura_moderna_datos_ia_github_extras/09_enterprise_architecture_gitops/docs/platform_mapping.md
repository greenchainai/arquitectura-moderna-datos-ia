| Ingesta desde ERP/API | Batch, incremental, CDC | Fabric Data Factory, Python, Databricks |

| Procesamiento masivo | Lakehouse, Spark | Databricks, Fabric Spark |
| Serving analítico | Gold, semantic layer | Snowflake, Fabric, Power BI |
| IA empresarial | RAG, agentes | Foundry, vector store, SQL endpoints |
| Control de cambios | GitOps, DataOps | GitHub, Azure DevOps, CI/CD |
| Infraestructura | IaC | Terraform, Bicep |
