# API tradicional vs Data Space

| Aspecto | API tradicional | Data Space |
|---|---|---|
| Foco | Intercambio técnico | Intercambio gobernado |
| Descubrimiento | Manual o documental | Catálogo |
| Condiciones de uso | Fuera de la API | Políticas asociadas |
| Contrato | Normalmente externo | Negociación/contrato técnico |
| Soberanía | Limitada | Principio central |
| Interoperabilidad | Caso a caso | Reglas comunes |
| Auditoría | Depende de implementación | Parte del diseño |
| Confianza | Bilateral | Federada |

Las APIs siguen siendo útiles, pero el data space añade gobierno, contrato, soberanía, auditoría y confianza federada.
